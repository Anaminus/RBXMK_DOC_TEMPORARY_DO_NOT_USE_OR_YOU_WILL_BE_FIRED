The program library is loaded directly into the global environment. It contains values related to the compiled executable.

ItemKindDescription
_RBXMK_VERSIONField

The current version of rbxmk.

Fields

_RBXMK_VERSION

_RBXMK_VERSION: string [readonly]

The _RBXMK_VERSION global variable is defined as a string containing the current version of rbxmk, formatted according to semantic versioning.
