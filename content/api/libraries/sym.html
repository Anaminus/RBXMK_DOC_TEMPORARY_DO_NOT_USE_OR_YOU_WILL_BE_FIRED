The sym library contains Symbol values. A symbol is a unique identifier that can be used to access certain metadata fields of an Instance.

An instance can be indexed with a symbol to get a metadata value in the same way it can be indexed with a string to get a property value:

local instance = Instance.new("Workspace")
instance[sym.IsService] = true
print(instance[sym.IsService]) --> true

The following symbols are defined:

$FIELDS

ItemKindDescription
AttrConfigField

Gets the inherited AttrConfig of an instance.

DescField

Gets the inherited descriptor of an instance.

IsServiceField

Determines whether an instance is a service.

MetadataField

Gets the metadata of a DataModel.

PropertiesField

Gets the properties of an instance.

RawAttrConfigField

Accesses the direct AttrConfig of an instance.

RawDescField

Accesses the direct descriptor of an instance.

ReferenceField

Determines the value used to identify the instance.
