The os library is an extension to the standard library. The following additional functions are included:

$FIELDS

ItemKindDescription
getenvFunction

Gets an environment variable.

Functions

getenv

os.getenv(name: string?): string?|{[string]: string}

The getenv function returns the value of the name environment variable, or nil if no such value is defined. If name is not specified, then a table of environment variable names mapped to values is returned.
